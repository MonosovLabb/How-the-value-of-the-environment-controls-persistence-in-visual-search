This Matlab code is for generating all figures and tables for the paper:

"How the value of the environment controls persistence in visual search"
by Traner, Bromberg-Martin, and Monosov (PLoS Computational Biology)

To do this, open Matlab, go to this directory ("Data and Figure Generators"), and run the following script:

Generate_All_Figures_And_Tables.m

Please note that the license in this directory applies only to code written by the authors. The code in the directory MatlabToolbox-2.8 (the IOSR toolbox) is governed by its own license located in that directory.
